# How to use
1. **Clone the repository**
    ```
    git clone https://github.com/JimKonstantinopoulos/Rate-n-Save.git
    ```
2. **Navigate to the project directory**
    ```
    cd Rate-n-Save
    ```
3. **Install dependencies**
    ```
    npm i
    ```
4. **Start the Fake Server**
    ```
    npm start
    ```
